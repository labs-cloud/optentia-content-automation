# App assets — REPLACE BEFORE SHIPPING

`icon.png`, `splash.png`, and `adaptive-icon.png` here are **1×1 placeholders** so
the project boots. Before an App Store build, replace them with real artwork:

- `icon.png` — 1024×1024, no alpha, no rounded corners (Apple rounds it).
- `splash.png` — centered logo on `#0A1322` (the Frosted Vapor navy canvas).

Tip: `npx @expo/image-utils` or the `expo-splash-screen` / EAS asset tooling can
generate the platform variants from a single source. App Store Connect also needs
a separate 1024×1024 marketing icon and iPhone screenshots.
