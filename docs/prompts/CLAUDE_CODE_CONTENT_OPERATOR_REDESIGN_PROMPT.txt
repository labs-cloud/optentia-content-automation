You are working inside the existing repository:

github.com/labs-cloud/optentia-content-automation

The current app already has content generation, queue, scheduling, publishing paths, analytics events, platform connections, media, and HeyGen support.

The goal is to redesign and refactor it into a premium, multi-client AI marketing operating system.

Do not rewrite the entire app from scratch unless absolutely necessary.
Preserve the existing React/Vite + Express/tRPC + Drizzle structure.

# High-Level Goal

Turn this from an Optentia-only content automation dashboard into a multi-client AI marketing operator with:

- Premium Holo/Linear/iOS-inspired design.
- Client/company workspaces.
- Brand Operating Profiles.
- Swipe-based brainstorm learning.
- Campaign-based content generation.
- Approval queue.
- Calendar scheduler.
- Analytics learning loop.
- Mobile-first architecture for future iOS app.
